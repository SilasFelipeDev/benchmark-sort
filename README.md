# benchmark-sort 

Um projeto de estudos em C voltado para a implementação, análise e comparação de algoritmos de ordenação.

O objetivo deste repositório não é servir como portfólio, mas como uma biblioteca pessoal de consulta e um laboratório para aprofundar conhecimentos em algoritmosm análise de complexidade e organização de projetos em C.

## Objetivos 
- Implementar diversos algortimos de ordenação do zero.
- Comparar desempenho entre algortimos.
- Medir tempo de execução.
- Contabilizar comparações e trocas.
- Estudar complexidade de tempo e espaço.
- Organizar o código em múltiplos arquivos seguindo boas práticas.

## Algoritmos
